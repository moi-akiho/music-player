Google Cloud 設定手順
  1. Google Cloud Console にアクセス
    console.cloud.google.com
  2. 「新しいプロジェクト」を作成（名前は何でもOK）
  3. 左メニュー →「APIとサービス」→「ライブラリ」→ Google Drive API を検索して「有効にする」
  4. 「APIとサービス」→「OAuth 同意画面」→「外部」で作成、アプリ名を入力してスコープに drive.readonly と drive.appdata を追加
  5. 「認証情報」→「＋認証情報を作成」→「OAuth クライアントID」→ アプリの種類：ウェブアプリケーション
  6. 「承認済みの JavaScript 生成元」に以下を追加：
    http://localhost:8086
  7. 作成後に表示される クライアントID をコピー
  8. gdrive.js の1行目 CLIENT_ID: 'YOUR_CLIENT_ID_HERE' の YOUR_CLIENT_ID_HERE の部分を貼り付けて保存

設定は一度だけです。タブレット買い替え後は同じURLを開くだけで使えます。
